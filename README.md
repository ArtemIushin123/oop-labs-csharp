# OOP Labs — C#

A collection of Object-Oriented Programming lab projects written in C# at ITMO University.

## Labs

### Lab 1 — Train Route Simulator
Simulation of automated magnetic levitation train routes.  
Key concepts: OOP fundamentals, encapsulation, immutability, Result pattern, unit tests

### Lab 2 — Message Distribution System
Corporate messaging system with flexible recipient routing and filtering.  
Key concepts: Structural patterns (Decorator, Composite), SOLID, mocking (NSubstitute)

### Lab 3 — Fantasy Battle Simulator
Turn-based battle engine for fantasy creatures with modifiers and spells.  
Key concepts: Creational & behavioral patterns (Factory, Decorator), OCP, unit + functional tests

### Lab 4 — File System Manager
Console application for navigating and managing a file system via CLI commands.  
Key concepts: SOLID, Command pattern, lazy loading, extensible flag-based CLI

### Lab 5 — ATM Web API
ATM simulation system with session-based authorization and account management.  
Key concepts: Hexagonal architecture, ASP.NET Core, Dependency Injection (MSDI), in-memory repositories, NSubstitute

## Tech Stack
C# · .NET · ASP.NET Core · NSubstitute · xUnit
